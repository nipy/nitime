Release notes for nitime version 0.5¶

Summary of changes¶

Major changes introduced in version 0.5 of nitime:

  1. Python 3 support. Work led by Paul Ivanov and Ariel Rokem, with help from Thomas Kluyver, and Matthew Brett.
  2. Multi-taper F-test for harmonic components. Work led by Mike Trumpis.
  3. Continuous integration testing with Travis. Work led by Ariel Rokem.
  4. Various fixes and robustifications from several contributors (see below).

Contributors to this release¶
